Complexity vs Coverage

This scatter plot shows the relationship between complexity and coverage for each class in the project, similar to that provided by Clover Hover over the bubbles to see the package and class name.

Why is this useful? Ideally, we want to increase test coverage and reduce complexity, so put your attention on the bubbles towards the top-left corner, and work your way down towards the bottom-right. This plot gives us ideas to explore. Note that this plot uses instruction coverage as opposed to line or branch coverage.
